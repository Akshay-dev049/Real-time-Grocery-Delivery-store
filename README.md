# Real-time-Grocery-Delivery-store
FreshGrocery — A real-time grocery delivery web app built with Python, Flask &amp; SQLite. Features 3-role system (Admin, Delivery, Customer), live GPS order tracking via Leaflet.js &amp; OSRM, Razorpay payment, 3km delivery radius enforcement, stock management, and security question-based password recovery. Deployed on PythonAnywhere.
